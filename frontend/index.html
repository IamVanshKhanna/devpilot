Frequently Asked Questions

Everything you need to know about DevPilot.

Does DevPilot replace human code review?

No. DevPilot handles the first pass — catching bugs, style issues, and anti-patterns. Humans still review for architecture, design, and business logic.

Is my source code safe?

Yes. Diffs are processed in-memory and never stored. We use read-only GitHub OAuth with minimal permissions. No source code leaves your repository.

What languages does it support?

All languages that GitHub supports. The AI model understands 50+ programming languages including Python, JavaScript, TypeScript, Go, Rust, Java, and more.

Can I customize the review rules?

Yes. Starter includes standard rules for bugs, security, and style. Pro lets you define custom patterns. Enterprise includes custom model training on your codebase.

How fast are the reviews?

Most PRs receive their first review comments within 60-120 seconds of opening. Complex PRs may take up to 3 minutes.

Is there a free trial?

Yes. Starter and Pro plans include a 14-day free trial. Open source repos are always free. No credit card required for the free tier.
