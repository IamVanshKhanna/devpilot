Frequently Asked Questions

Does DevPilot replace human code review?

No. DevPilot handles the first pass — catching bugs, style issues, and anti-patterns. Humans still review for architecture, design, and business logic. Think of it as an automated first reviewer that makes the human review faster and more focused.

Is my source code safe?

Absolutely. Diffs are processed entirely in-memory and never written to disk. We use read-only GitHub OAuth tokens with the minimum permissions needed. No source code leaves the processing pipeline. After review, all diff data is discarded.

What languages does DevPilot support?

DevPilot supports all languages that GitHub recognizes — over 50 programming languages including Python, JavaScript, TypeScript, Go, Rust, Java, C++, Ruby, PHP, Swift, Kotlin, and many more. The AI model has been trained on a broad corpus of code.

Can I customize the review rules?

Yes. Starter includes standard rules for bugs, security, and style. Pro lets you define custom patterns using YAML configuration. Enterprise includes custom model training on your proprietary codebase for maximum relevance.

How fast are reviews?

Most PRs receive their first review comments within 60-120 seconds of opening. Complex PRs or very large diffs may take up to 3 minutes. We're constantly optimizing the AI pipeline for speed.

Do you offer a free trial?

Yes! Starter and Pro plans include a 14-day free trial with no credit card required for the first 7 days. Open source repositories are always free — no limits, no catches.

How does team learning work?

When your team accepts or dismisses DevPilot's suggestions, the system learns. Accepted patterns are reinforced. Rejected ones are deprioritized. Over time, DevPilot's reviews align more closely with your team's actual conventions.

What permissions does the GitHub App need?

Read-only access to pull requests and repository metadata. Write access to post review comments. That's it. DevPilot cannot read your source code outside of pull requests or modify any repository settings.
